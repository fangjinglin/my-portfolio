智慧停車管理系統 (Django)

技術：Python, Django, HTML, CSS, JS, SQLite, Bootstrap

  • 停車紀錄管理與統計分析
  • 自動計算停車費率，依時段與車型收費
  • 資安防護：密碼加密、權限控管、SQL 注入防護、CSRF 防護

ArtSpace (Kotlin Compose)

技術：Kotlin, Jetpack Compose, Material 3

行事備忘錄 APP (Android)

技術：Kotlin, Android Jetpack Compose, Room Database, Google Maps API

  • 首頁顯示當日備忘錄清單，含時間、地點與說明
  • 備忘錄可點擊地點顯示地圖，增強行程管理效率
  • 支援新增與刪除備忘錄，刪除需確認，新增需檢查標題與時間格式
  • 浮動按鈕快速新增備忘錄，日期固定為當天
  • 到達備忘錄時間時推播提醒，方便即時提醒工作
